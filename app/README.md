# cabulous app

Frontend do Cabulous Site.

Este diretorio sera usado para a aplicacao web em Next.js do ecossistema Cabulous.

## Objetivo

O `app` concentrara a interface do site, consumindo os recursos expostos pelo backend em `service`.

## Stack planejada

- Next.js
- TypeScript
- Prettier

## Status

No momento, este diretorio esta reservado para a implementacao do frontend.
