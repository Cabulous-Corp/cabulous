# Cabulous Web

Next.js 16+ application for the Cabulous Site management.

## Getting Started

Para configurar o projeto pela primeira vez, utilize o `Makefile`:

```bash
# Copia o .env de exemplo e instala as dependências
make setup

# Inicie o servidor de desenvolvimento
make dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## Scripts Disponíveis

- `make dev` — Inicia o servidor de desenvolvimento
- `make build` — Gera o build de produção
- `make install` — Instala as dependências (bun)
- `make lint` — Executa o linting
- `make setup` — Configura o ambiente inicial
